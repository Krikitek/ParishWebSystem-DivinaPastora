Proseso ng Paghiling

1. Punan ang Form

Kumpletuhin ang online form para sa iyong piniling sertipiko. Siguraduhing tama ang lahat ng impormasyon.

2. Beripikasyon

Ang staff ng parokya ay magbeberipika ng iyong impormasyon sa mga rekord ng simbahan.

3. Paghahatid

Piliin kung paano mo gustong matanggap ang iyong sertipiko: pickup o delivery.

4. Pagbabayad

Magbayad ng kinakailangang bayarin sa pamamagitan ng mga aprubadong payment channels.

5. Paghahanda

Ang iyong sertipiko ay ihahanda at pipirmahan ng awtorisadong tauhan ng parokya.
